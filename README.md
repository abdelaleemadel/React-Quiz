## Installation

1. Clone the repository:

```bash
git clone https://github.com/gjonhajdari/react-quiz.git
```

2. Navigate to the project directory:

```bash
cd React-Quiz
```

3. Install dependencies:

```bash
npm install
```

3. Run the JSON server(Optional simulated API):

```bash
npm run server
```

4. Start the React app:

```bash
npm start
```
